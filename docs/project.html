Berkeley Admission Quiz App

A dumb introductory application to android development filled with the hate that Stanford has for Berkeley. Joke aside, the application asks the user which of the two number buttons present on the screen is greatest. The app keeps track of points gained and lost during the play session by dynamically updating the TextView class.

To Do List

A robust applicaton that permits the user to save tasks within an ArrayList without there beining and data losses when the application gets terminated. The ArrayList is saved and is accesible for use during the next session. The ArrayList is hooked to an ArrayAdapter which is set to the listView for the user to observe the data that has been saved or added to the ArrayList.
